TensorFlow의 작동 방법은 다음과 같습니다. [^basic_usage]

* computations 은 graphs 로 표현합니다.
* graphs 는 Sessions 의 context 에서 실행합니다.
* data 는 tensors 로 표현합니다.
* Variables 로 상태를 유지합니다.
* feeds 와 fetches 를 사용해서 data 를 임의의 작업에 넣거나 가져옵니다.
