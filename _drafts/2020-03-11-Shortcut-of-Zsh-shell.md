참고한 블로그에 나온 단축키와 좀 다른 것들이 있는 것 같습니다.

### 커서 이동하기

일단 저의 경우 `alt` + `left` / `right` 가 커서를 맨 앞/뒤로 이동하는 것을 확인하였습니다.

이것이 `ctl` + `A` / `E` 조합보다 편한 것 같습니다.

### 뒤로 지우기

맥에서는 `fn` + `delete` 로 한글자씩 지울 수 있습니다.[^delete-backwards]
