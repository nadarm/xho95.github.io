Goole Search Console 에서 DNS record 는 따로 DNS 가 있는 경우 사용하는 것입니다.[^google-domain] 즉, 따로 도메인 이름이 있을 때 쓰는 기능입니다. 다만 좀 더 확인 필요합니다.

HTML tag 방식을 사용할 수 있으며, 이 때는 `_config.yml` 파일에서 전체 적용 가능합니다.
